## Technische Details

Dieses Projekt konvertiert das PDF-Buch in AsciiDoc-Format und generiert daraus HTML.

**HTML generieren:**
```bash
asciidoctor index.adoc
```

Die generierte `index.html` enthält die vollständige Dokumentation mit allen Bildern als externe Referenzen.
